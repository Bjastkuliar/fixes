##How to Update NodeJS on Linux

###Clear Node Pacakge Manager Cache

`sudo npm cache clean -f`

###Install n globally

`sudo npm install -g n`

###Use n to upgrade to latest stable version

`sudo n stable`

or

`sudo n 0.10.18`

###Check node Version

`node -v`
